Clases

Muchas veces cuando creamos un objeto, estamos creando una Plantilla. En lugar de copiar esa Plantilla una y otra vez, Javascript nos da acceso a lo que llamamos un constructor o CLASS. Estas son utiles para crear muchos objetos que comparten algunas de las mismas propiedades y metodos.

Funcion constructora

Prototype

La creacion de funciones es costosa y cada vez que creamos un nuevo objeto de clase con metodos, estamos recreando estos metodos en la memoria. Si creariamos miles de objetos de clase a partir de una clase con docenas de metodos, la memoria se acumularia rapidamente. Las clases tienen una forma unica de establecer un metodo una vez y dar acceso a cada objeto de esa clase a esos metodos. Esto se llama Protoype, una propiedad que posee cada clase.
El metodo CREATE de los objetos nos permite crear un nuevo objeto a partir de un prototype especificado.
El metodo ASSIGN de los objetos permite agregar propiedades a un objeto pasado por parametro
